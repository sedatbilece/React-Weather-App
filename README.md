# React Weather App

## Description

In this project, dynamic weather display was made using www.weatherapi.com

### Features
<ul>
  <li>City search</li>
  <li>Dynamic Icon</li>
  <li>Not Found Modal</li>
  <li>Extra Info</li>
</ul>
